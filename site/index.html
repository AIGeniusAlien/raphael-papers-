Raphael Beta

A Clinical Copilot for Multimodal AI Decision Support

GitHub Docs Repository Root

Safety • Evidence • Speed

Raphael Beta combines LLM reasoning with guideline-grounded retrieval, deterministic guardrails, and SMART-on-FHIR integration to help clinicians reason faster—while always keeping humans in the loop.

Preprint (soon) View Code

Multimodal

Text, images (DICOM/JPEG), and voice. Evidence is cited and explainable.

Safety Guardrails

Dose limits, interaction checks, uncertainty and abstention logic.

EHR-Ready

SMART-on-FHIR OAuth, CRUD for Patient/Observation/DocumentReference, and provenance.

Reproducible

Scripts, seeds, and configs for RAG, evals, and safety test suites.
